PROJET 7 - Créer un réseau social d'entreprise

Executer le projet

1) Cloner le projet en local
2) Créer un dossier images dans le dossier projet_7
3) Lancer l'invite de commande dans le dossier projet_7
4) Lancer npm install

Mettre en place la base de données pour exécuter le projet

Pré-requis: MySQL installé en local

1) Lancer MySQL et créer une base de données

2) Ouvrir le fichier db.js (path : Projet_7\backend\config\db.js)

3) Remplir la constante sequelize avec les informations de votre base de données : 

const sequelize = new Sequelize(NOM_DE_LA_BD, USERNAME, PASSWORD, 




Lancer le projet

Lancer index.html dans Projet 7\views
